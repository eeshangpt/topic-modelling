# TOPIC MODELLING.

## Create Data Directory.

Steps:

- `mkdir data`
- `mkdir -p data/corpus`
- `mkdir -p data/embedded`
- Extract all the documents in `./data`. (**EXPERIMENTAL**)
    - `dvc pull`
- If the above steps fails, please raise an issue on this repository.

## Create Logs Directory.

Steps:

- `mkdir logs`

## Create Results Directory.

Steps:

- `mkdir res`
